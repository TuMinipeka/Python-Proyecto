### 11. Calculadora de Porcentaje de Ganancia

- **Enunciado:** Escribe un programa que solicite al usuario el **costo** de un producto y su **precio de venta**. El programa debe calcular y mostrar cuál es el porcentaje de ganancia obtenido sobre el costo.

- **Resultado en consola:**

  ```
  Ingresa el costo del producto: 50
  Ingresa el precio de venta: 75
  La ganancia es del 50.0%
  ```
  
- **Pistas:** Recuerda convertir las entradas a `float()`. Realiza primero la resta para saber la ganancia monetaria y luego la división. Puedes usar `f-strings` para mostrar el símbolo `%`.

  Para realizar este ejercicio, debes traducir la siguiente lógica matemática a código Python:

  1. **Obtener la ganancia neta:** Restas el costo al precio de venta.

     Ganancia=Precio Venta−Costo

  2. **Obtener la proporción:** Divides esa ganancia entre el costo original.

     Proporcion=CostoGanancia

  3. **Convertir a porcentaje:** Multiplicas el resultado por 100.

     %Ganancia=Proporcion×100
  
  ```python
  precio = int(input("Ingrese el precio actual del producto: "))
  
  porcentaje = int(input("Ingrese el porcentaje de ganancia del precio del producto(1 a 100):"))
  
  # porcentaje = 30% -> 1 - 0.3 - Sobre el precio del producto
  
  nuevo_precio = precio / (1 - porcentaje / 100)
  
  valor_porcentaje = nuevo_precio * (porcentaje / 100)
  
  
  print(f"Precio actual: {precio}\nPorcentaje: {porcentaje}%\tGanancia: {valor_porcentaje}")
  print(f"El nuevo precio del producto es de: {nuevo_precio}")
  print(f"El porcentaje de ganancia es de: {(nuevo_precio * (porcentaje/100))}")
  ```
